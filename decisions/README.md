# Decisions (ADRs)

Cross-cutting architecture/technical decisions that span the Sodax product (more
than one repo). Repo-scoped decisions live under `projects/<repo>/` instead.

- One file per decision: `NNNN-short-title.md` (e.g. `0001-intent-settlement.md`).
- Start from `0000-adr-template.md`.
- Never delete a superseded ADR — mark it `Superseded by NNNN` and keep it.
